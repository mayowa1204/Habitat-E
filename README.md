
# new-app

## Prerequisites

- Install Node: https://nodejs.org/en/download/
```bash
$ cd new-app
```
Within the new-app folder:

```bash
$ npm install
```

### Development

To compile and hot-reload the frontend

```bash
# new-app in watch-mode
$ npm run serve


```

### Production

```bash
# new-app
$ npm run build
